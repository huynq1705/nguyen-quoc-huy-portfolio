01 — Design Direction

Ý tưởng thiết kế portfolio

Kết hợp DNA thị giác từ 2 hệ sinh thái bạn đã xây dựng: gradient tím–indigo của Smart Slide và dark enterprise của MRAG — tạo portfolio cá nhân độc đáo, không giống template generic.

🎨

Dark + Gradient Accent

Nền tối sâu (#06060f) với gradient tím → xanh → cyan. Glassmorphism nhẹ, glow orb ambient — lấy cảm hứng từ Smart Slide hero và MRAG User dark mode.

📐

Bento Grid Layout

Layout dạng bento box hiện đại — mỗi project là một "tile" với kích thước khác nhau theo mức độ quan trọng. Responsive, scroll-driven reveal animation.

⚡

Micro-interactions AI

Typewriter effect, streaming dots, particle float — những pattern bạn đã implement trong creating screen & chat streaming, dùng làm signature cho portfolio.

🧩

Case Study sâu

Mỗi project có trang riêng: Problem → Solution → Tech highlights → Screenshots → Lessons learned. Không chỉ show ảnh mà kể story kỹ thuật.

🌐

Bilingual VI/EN

Toggle ngôn ngữ — phản ánh kinh nghiệm i18n thực tế (EN/JA) từ các project. Mặc định tiếng Việt, EN cho recruiter quốc tế.

📱

Mobile-first

Responsive từ mobile → ultrawide (5xl breakpoint như MRAG User). Sidebar collapse, touch-friendly, gesture-aware.

📋 Cần bạn xác nhận trước khi build

  1. Tên hiển thị — Tên đầy đủ / nickname cho hero section?
  2. Ảnh cá nhân — Có muốn đặt avatar/photo không, hay chỉ dùng typography?
  3. Ngôn ngữ mặc định — Tiếng Việt, English, hay bilingual toggle?
  4. Thông tin liên hệ — Email, GitHub, LinkedIn, phone?
  5. CV/Resume — Có file PDF sẵn để link download không?
  6. Case study depth — Mỗi project 1 trang riêng hay tất cả trên single page?
  7. Screenshots — Bạn có thể cung cấp screenshot từ các project (hoặc tôi dùng placeholder)?
  8. NDA — Các project enterprise có cần anonymize tên công ty không?
  9. Tech stack build — Next.js (như Smart Slide) hay Vite + React (như MRAG Admin)?
  10. Hosting — Vercel, GitHub Pages, hay self-host?
